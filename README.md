# JARVIS---AI-VOICE-ASSISTANT-
Python-based offline AI voice assistant using local LLM (Phi) and NewsAPI

Jarvis is a **Python-based AI voice assistant**, similar to **Alexa**, built as a student project to understand how real voice assistants work by integrating **AI, voice recognition, APIs, and system automation**.

The assistant uses a **free local AI model (Phi)**, making it **offline-capable**, **privacy-friendly**, and **cost-free** ‚Äî perfect for learning and experimentation without paid APIs.
